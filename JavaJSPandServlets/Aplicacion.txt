En este proyecto se hara una pagina Web implementando la tecnologia de Java, JSP que es una herramienta que
nos permite utilizar HTML y Java dentro de un mismo documentos y haremos el Backend del servicio con
Servlets y Tomcat.

El ciclo de vida de Servlets para crear la web app es cargar la clase servlets, crear la instancia de 
servlet, hacer el init con servlet, llamar el servicio de servlets service() method and destroy() method

Investigacion JSP, servlet and Tomcat o maven for web page
